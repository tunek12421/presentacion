Proyecto 2

AcademyUpds
Plataforma de Cursos

Plataforma Web para la Gestión y Difusión de Cursos de Formación Continua

Proyecto de Desarrollo de Software

El Problema

Información Dispersa
Los cursos de formación se promocionan por canales separados sin un punto centralizado.
Difícil Acceso
Los estudiantes no encuentran fácilmente información detallada sobre cursos disponibles.
Sin Organización
No existe categorización por facultades ni filtrado por áreas de conocimiento.
Experiencia Limitada
La información de cursos no incluye detalles como módulos, instructores ni prerequisitos.

La Solución

Plataforma web tipo catálogo que centraliza y organiza los cursos de formación continua por facultades y academias.

Características

  • Catálogo organizado por facultades
  • Fichas detalladas de cada curso
  • Información de instructores
  • Módulos, prerequisitos y FAQs

Tecnologías

  • Build Tool: Vite
  • Frontend: JavaScript (SPA)
  • Estilos: Tailwind CSS
  • Deploy: Vercel

Organización de Cursos

Por Facultades

  • Ciencias de la Salud
  • Ingeniería
  • Ciencias Empresariales
  • Ciencias Jurídicas

Por Academias

  • Academia MikroTik (Certificación MTCNA)
  • Cursos especializados por área
  • Programas de formación continua

Cursos Disponibles

MikroTik MTCNA Primeros Auxilios Excel Experto Tributación Aplicada Litigación Manejo de Cadáveres

Arquitectura del Sistema

Frontend

Vanilla JS + Vite

  • SPA con routing cliente
  • Componentes modulares
  • Carga dinámica de contenido
  • Sin frameworks pesados

Estilos

Tailwind CSS

  • Utility-first CSS
  • Diseño responsive
  • Tema personalizado UPDS
  • FontAwesome iconos

Infraestructura

Vercel

  • Deploy automático
  • CDN global
  • SSL/HTTPS incluido
  • Optimización de assets

Estructura del Proyecto

Módulos JavaScript

  • router.js - Motor de rutas SPA
  • data.js - Datos de cursos y estado
  • components.js - UI reutilizable
  • app.js - Lógica de renderizado

Rutas Disponibles

  • / - Página principal
  • /facultad/:nombre - Cursos por facultad
  • /academia/:nombre - Cursos por academia
  • /curso/:id - Detalle del curso

Funcionalidades Destacadas

SPA Sin Recargas

Navegación fluida sin recargar la página. Router personalizado que maneja todas las rutas del lado del cliente.

Fichas de Curso Completas

Cada curso muestra instructor, biografía, módulos, duración, prerequisitos, FAQs y habilidades a desarrollar.

Diseño Responsive

Interfaz adaptable a móviles, tablets y escritorio. Menú hamburguesa para dispositivos pequeños.

Carga Optimizada

Skeleton loaders durante la carga, lazy loading de imágenes y assets optimizados con Vite.
