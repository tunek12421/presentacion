Sistema Web para la Gestión de Inventario y Préstamos de Biblioteca

Proyecto de Desarrollo de Software

El Problema

Registro Manual
Gestión mediante hojas de cálculo y registros físicos, consumiendo tiempo y generando errores.
Sin Trazabilidad
No existe historial de préstamos ni auditoría de movimientos de libros.
Sin Control de Roles
Cualquier persona puede modificar registros sin control de acceso diferenciado.
Pérdida de Información
Riesgo de extravío de datos por falta de respaldos y centralización.

La Solución

Sistema web que automatiza el inventario de libros y gestión de préstamos con control de roles y trazabilidad completa.

Características

  • CRUD de libros y categorías
  • Gestión de préstamos y devoluciones
  • Control de roles (Bibliotecario/Becario)
  • Historial y auditoría

Tecnologías

  • Backend: Django + PostgreSQL
  • Frontend: React + Vite
  • Auth: JWT + RBAC
  • Deploy: Red local

Arquitectura del Sistema

Backend API

Django REST

  • API REST documentada
  • PostgreSQL + ORM
  • Autenticación JWT
  • Validación de datos

Frontend

React + Vite

  • SPA moderna
  • Componentes reutilizables
  • Gestión de estado
  • Diseño responsive

Base de Datos

PostgreSQL

  • Modelo relacional
  • Integridad referencial
  • Índices optimizados
  • Respaldos automáticos

Control de Roles (RBAC)

Bibliotecario

  • Gestión completa de inventario
  • Registro de préstamos y devoluciones
  • Editar y eliminar registros
  • Ver historial por fechas
  • Búsqueda por CI y reconocimiento facial

Becario

  • Registro de préstamos y devoluciones
  • Consulta de inventario (solo lectura)
  • Ver historial del día actual
  • Búsqueda básica por CI
  • Sin permisos de edición/eliminación

Módulos del Sistema

Gestión de Inventario

Alta, baja y modificación de libros. Categorización, búsqueda avanzada y control de ejemplares disponibles.

Gestión de Préstamos

Registro de préstamos y devoluciones. Identificación por CI o reconocimiento facial del docente.

Historial y Auditoría

Registro de todas las operaciones con fecha, hora y usuario. Filtrado por rangos de fechas.

Gestión de Usuarios

Administración de bibliotecarios y becarios. Asignación de permisos según rol.
