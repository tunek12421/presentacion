Sistema Web para la Generación Automatizada de Certificados desde Archivos Excel

Proyecto de Desarrollo de Software

El Problema

Proceso Manual
Crear certificados uno por uno en Word/PDF requiere horas de trabajo repetitivo.
Errores de Transcripción
Copiar nombres manualmente genera errores tipográficos y omisiones.
Inconsistencia
Diferentes formatos, fuentes y posiciones en cada certificado generado.
Escalabilidad
Cursos con 100+ participantes hacen el proceso manual insostenible.

La Solución

Aplicación web que genera certificados en lote desde archivos Excel, con detección inteligente de datos y plantillas personalizables.

Características

  • Detección automática de columnas
  • Plantillas PDF personalizables
  • Vista previa antes de generar
  • Descarga masiva en ZIP

Tecnologías

  • Backend: Flask (Python)
  • Excel: Pandas + OpenPyXL
  • PDF: ReportLab + PyPDF2
  • Deploy: Docker

Arquitectura del Sistema

Backend

Flask + Python

  • Factory Pattern
  • Blueprints modulares
  • Capa de servicios
  • API REST endpoints

Procesamiento

Pandas + ReportLab

  • Lectura Excel inteligente
  • Detección de encabezados
  • Generación PDF vectorial
  • Merge con plantillas

Frontend

HTML/CSS/JS

  • Interfaz responsiva
  • Tema claro/oscuro
  • Preview en tiempo real
  • Controles de ajuste fino

Detección Inteligente de Excel

Análisis Automático

  • Detecta fila de encabezados
  • Identifica columna de nombres
  • Identifica columna de CI/ID
  • Limpia datos vacíos

Palabras Clave Buscadas

  • Nombres: nombre, participante, estudiante, alumno
  • ID: ci, cédula, carnet, documento
  • Análisis de longitud y unicidad
  • Detección de patrones numéricos

Funcionalidades Destacadas

Plantillas PDF

Sube tu diseño de certificado como PDF y el sistema superpone los nombres en la posición exacta que configures.

Fuentes Personalizadas

12+ fuentes incluidas (Helvetica, Times, Courier) más soporte para cargar fuentes TTF/OTF propias.

Vista Previa

Genera un certificado de muestra con el primer registro para verificar posición, tamaño y estilo antes de procesar todo.

Generación Masiva

Procesa cientos de certificados y los comprime en un único archivo ZIP para descarga inmediata.

Estructura del Proyecto

Organización Modular

app/
├── __init__.py      # Factory Pattern
├── config.py        # Configuración
├── routes/          # Blueprints
│   ├── upload.py
│   ├── certificate.py
│   └── color.py
├── services/        # Lógica de negocio
│   ├── excel_service.py
│   ├── pdf_service.py
│   └── font_service.py
└── templates/
    └── index.html

Endpoints API

  • POST /upload_excel - Subir archivo Excel
  • POST /upload_template - Subir plantilla PDF
  • POST /upload_font - Subir fuente TTF
  • POST /certificate/preview - Generar vista previa
  • POST /certificate/generate - Generar todos
  • GET /certificate/download - Descargar ZIP
