Sitio Web para el 2° Congreso Internacional de Inteligencia Artificial UPDS

Proyecto de Desarrollo de Software

El Problema

Difusión Limitada
Los eventos académicos no cuentan con plataformas profesionales para promoción y registro.
Gestión Manual
El registro de participantes y venta de tickets se realiza de forma dispersa y poco eficiente.
Información Fragmentada
Cronogramas, ponentes y detalles del evento no están centralizados en un solo lugar.
Sin Interactividad
Los asistentes no tienen forma de resolver dudas o interactuar antes del evento.

La Solución

Portal web interactivo que centraliza información, ponentes, cronograma, venta de tickets y soporte mediante chatbot para el congreso de IA.

Características

  • Carrusel de 13 ponentes internacionales
  • Cronograma interactivo de 12 horas
  • Sistema de tickets integrado
  • Chatbot de soporte

Tecnologías

  • Frontend: HTML5 + CSS3 + JS ES6
  • Datos: JSON dinámico
  • Iconos: Font Awesome 6.0
  • APIs: Intersection Observer

Arquitectura del Sistema

Frontend

Vanilla JS ES6

  • Componentes modulares
  • ES6 Modules (.mjs)
  • Intersection Observer
  • PWA-ready

Estilos

CSS3 Modular

  • Variables CSS
  • Arquitectura por secciones
  • Responsive design
  • Animaciones smooth

Datos

JSON + Cache

  • config.json (evento)
  • speakers.json (ponentes)
  • schedule.json (cronograma)
  • Fallback offline

Componentes Principales

Speaker.js

  • Carrusel infinito (5 copias)
  • Responsive (1-4 cards)
  • ResizeObserver adaptativo
  • Auto-scroll bidireccional
  • Categorías: Internacional / Nacional

Schedule.js + Tickets

  • 18 sesiones en 12 horas
  • Tipos: keynote, panel, workshop, webinar
  • Integración streaming (YouTube, Zoom)
  • Tickets: UPDS 90 Bs / Externos 120 Bs
  • Múltiples pasarelas de pago

Funcionalidades Destacadas

Carrusel de Ponentes

13 expertos internacionales y nacionales en IA. Scroll infinito con animaciones suaves y modales de detalle.

Chatbot Interactivo

Asistente virtual para resolver dudas sobre fecha, ubicación, ponentes, tickets y participación en el evento.

Webinars en Vivo

Integración con YouTube, Facebook, LinkedIn y Zoom. Chat interactivo, Q&A y salas de networking.

Accesibilidad

ARIA labels, navegación por teclado, subtítulos, descripción de audio y soporte para preferencias de movimiento reducido.
